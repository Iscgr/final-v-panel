# 🌱 راهنمای شروع سریع MarFaNet

این سند مخصوص کسانی است که اولین بار می‌خواهند MarFaNet را روی یک سرور تازه راه‌اندازی کنند. اگر تاکنون با Docker یا لینوکس کار نکرده‌اید، نگران نباشید؛ مراحل را قدم‌به‌قدم دنبال کنید.

---

## 🧭 MarFaNet چیست؟
MarFaNet یک پنل تحت‌وب برای مدیریت نمایندگان، فاکتورها و گزارش‌های مالی است که همه اجزای لازم (اپلیکیشن، پایگاه‌داده، کش، وب‌سرور و SSL) را خودش نصب می‌کند. تنها کاری که باید انجام دهید اجرای یک اسکریپت است؛ بقیه کارها خودکار انجام می‌شود.

---

## ✅ قبل از شروع
| مورد | توضیح |
|------|-------|
| سیستم‌عامل | Ubuntu 20.04 یا 22.04 یا 24.04 (کاربر روت یا sudo) |
| دسترسی | اتصال SSH به سرور و دسترسی اینترنت |
| دامنه | رکورد `A` باید به IP سرور اشاره کند (برای گواهی SSL) |
| پورت‌ها | پورت‌های 80 و 443 باید باز باشند |
| منابع | حداقل 2 هسته CPU، 4 گیگ RAM و 15 گیگ فضای خالی |

> اگر هنوز دامنه ندارید می‌توانید نصب را انجام دهید، اما SSL صادر نمی‌شود و باید بعداً آن را تکرار کنید.

---

## ۱. نصب خودکار (۳ گام ساده)
1. از طریق SSH وارد سرور شوید.
2. فایل نصب را دانلود کنید:
   ```bash
   curl -sSL https://raw.githubusercontent.com/Iscgr/final-v-panel/prof/scripts/auto-install.sh -o auto-install.sh
   chmod +x auto-install.sh
   ```
3. اسکریپت را با دسترسی روت اجرا کنید:
   ```bash
   sudo ./auto-install.sh
   ```

### اگر دامنه اختصاصی خودتان را دارید
قبل از اجرای اسکریپت، نام دامنه را به شکل زیر تعیین کنید:
```bash
sudo DOMAIN=your-domain.com ./auto-install.sh
```
- اسکریپت Docker، PostgreSQL، Redis و Nginx را نصب و پیکربندی می‌کند.
- گواهی SSL (Let’s Encrypt) به‌صورت خودکار درخواست می‌شود.
- تمام رمزهای مهم تولید و در فایل `.env` ذخیره می‌شوند.

> نصب کامل معمولاً 5 تا 10 دقیقه طول می‌کشد. هنگام اجرا صفحه را نبندید.

---

## ۲. بررسی نتیجه نصب
پس از پایان اسکریپت، خروجی شبیه نمونه زیر است:
```
============================================================
 نصب MarFaNet تکمیل شد
============================================================
دامنه: https://example.com
پنل:  https://example.com/admin
پورتال: https://example.com/portal/[ID]
Admin User: admin
Admin Pass: <RAND_PASS>
...
============================================================
```

### اطلاعات مهم بعد از نصب
| مورد | مسیر / دستور |
|------|---------------|
| آدرس پنل مدیریت | `https://YOUR-DOMAIN/admin` |
| رمز و نام کاربری ادمین | فایل `/opt/marfanet/.env` (متغیرهای `ADMIN_USERNAME`, `ADMIN_PASSWORD`) |
| فایل تنظیمات محیطی | `/opt/marfanet/.env` |
| فایل‌های Compose و Nginx | `/opt/marfanet/docker-compose-stack.yml` و `/opt/marfanet/nginx.conf` |
| ابزار مدیریت | دستور `agent` |

> پیشنهاد می‌شود در اولین ورود، رمز کاربر `admin` را تغییر دهید.

---

## ۳. آشنایی با ابزار `agent`
اسکریپت نصب یک دستور ساده به نام `agent` می‌سازد. کافی است بنویسید:
```bash
agent
```
و یکی از گزینه‌ها را انتخاب کنید:
```
1) بکاپ + ارسال تلگرام
2) آپدیت پنل
3) ریستارت پنل
4) نمایش لاگ
5) وضعیت سلامت
q) خروج
```

### دستورهای پرکاربرد
| فرمان | کارکرد |
|--------|--------|
| `agent backup` | تهیه نسخه پشتیبان از دیتابیس (فایل در `/opt/marfanet/backups`) |
| `agent update` | دریافت آخرین کدها و بازسازی کانتینر اپلیکیشن |
| `agent restart` | راه‌اندازی مجدد تمام سرویس‌ها |
| `agent logs` | نمایش 200 خط آخر لاگ اپلیکیشن |
| `agent health` | بررسی آدرس `/health` بدون نیاز به مرورگر |

اگر می‌خواهید بکاپ به تلگرام ارسال شود، شناسه ربات و چت را در فایل `.env` مقداردهی کنید:
```env
TELEGRAM_BOT_TOKEN=xxx
TELEGRAM_CHAT_ID=123456
```

## ۴. به‌روزرسانی یا نصب دوباره
اگر از قبل MarFaNet را نصب کرده‌اید و می‌خواهید نسخه جدید را بگیرید، کافی است اسکریپت تازه را دانلود و اجرا کنید؛ ابزار به‌طور خودکار مخزن را روی نسخه جدید تنظیم می‌کند.
```bash
curl -sSL https://raw.githubusercontent.com/Iscgr/final-v-panel/prof/scripts/auto-install.sh -o auto-install.sh
chmod +x auto-install.sh
sudo ./auto-install.sh
```
