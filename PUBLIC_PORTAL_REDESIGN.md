## ویژگی‌های طراحی جدید

### 🎨 طراحی بصری
- **رنگ‌بندی حرفه‌ای**: استفاده از پالت رنگی هماهنگ با تاکید بر آبی و بنفش
- **تم تاریک**: قابلیت تغییر به تم تاریک برای راحتی بیشتر کاربر
- **تایپوگرافی بهینه**: استفاده از فونت Vazirmatn برای خوانایی بهتر فارسی
- **انیمیشن‌های ظریف**: انتقال‌های نرم و انیمیشن‌های تعاملی

### 📱 ریسپانسیو
- **موبایل-محور**: طراحی ابتدا برای موبایل و سپس مقیاس‌پذیری به دسکتاپ
- **ناوبری هوشمند**: منوی متفاوت برای موبایل (پایین صفحه) و دسکتاپ (بالای صفحه)
- **چیدمان پویا**: Grid و Flexbox برای تطبیق خودکار با اندازه صفحه

### 🚀 عملکرد
- **لودینگ زیبا**: اسکلتون لودر برای تجربه روان‌تر
- **بهینه‌سازی رندر**: استفاده از CSS Variables برای تغییرات سریع تم
- **دسترسی‌پذیری**: ARIA labels و کیبورد ناوبری

### 🔧 قابلیت‌های کاربری

#### بخش داشبورد
- نمایش موجودی مالی
- نمایش بدهی کل
- نسبت پرداخت
- شاخص دقت

#### بخش دانلود اپلیکیشن‌ها
- نمایش اپلیکیشن‌های توصیه شده
- QR کد برای دانلود سریع
- دکمه دانلود مستقیم

#### بخش پرداخت‌ها
- جستجو در پرداخت‌ها
- مرتب‌سازی بر اساس تاریخ و مبلغ
- فیلترهای پیشرفته
- نمایش وضعیت هر پرداخت

#### بخش فاکتورها
- نمایش فاکتورها به ترتیب FIFO
- نمایش وضعیت هر فاکتور (پرداخت شده، معوقه، در انتظار)
- امکان مشاهده جزئیات

#### بخش اعلانات
- دریافت پیام‌ها از پنل ادمین
- فیلتر بر اساس نوع (اطلاعات، هشدار، خطا)
- نمایش پیام‌های خوانده نشده
- امکان علامت‌گذاری به عنوان خوانده شده

## نحوه استفاده

پورتال عمومی به صورت خودکار از طریق URL با پارامتر `publicId` در دسترس است:

```
/portal/:publicId
```

### مثال:
```
/portal/ABC123
```
